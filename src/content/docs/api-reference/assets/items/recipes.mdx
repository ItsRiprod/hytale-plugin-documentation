The recipe system defines crafting requirements for items. Recipes are embedded directly in item definitions and specify input materials, output items, crafting bench requirements, and processing time.

## Recipe Schema

### Core Properties

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `Input` | `array` | Yes | Required materials |
| `Output` | `array` | No | Produced items (defaults to containing item) |
| `BenchRequirement` | `array` | No | Required crafting stations |
| `TimeSeconds` | `float` | No | Crafting duration in seconds |
| `KnowledgeRequired` | `bool` | No | Requires recipe knowledge |

### Input Entry

| Field | Type | Description |
|-------|------|-------------|
| `ItemId` | `string` | Item ID to consume |
| `ResourceTypeId` | `string` | Resource type (e.g., `Fuel`) |
| `Quantity` | `int` | Amount required |

### Output Entry

| Field | Type | Description |
|-------|------|-------------|
| `ItemId` | `string` | Item ID to produce |
| `Quantity` | `int` | Amount produced (default: 1) |

### Bench Requirement

| Field | Type | Description |
|-------|------|-------------|
| `Id` | `string` | Crafting bench ID |
| `Type` | `string` | Requirement type |
| `Categories` | `array` | Recipe categories this bench supports |
| `RequiredTierLevel` | `int` | Minimum bench tier level |

## Requirement Types

| Type | Description |
|------|-------------|
| `Crafting` | Standard crafting bench |
| `DiagramCrafting` | Requires unlocked diagram |

## Crafting Benches

### Common Bench IDs

| Bench ID | Description | Common Categories |
|----------|-------------|-------------------|
| `Weapon_Bench` | Weapon crafting | `Weapon_Sword`, `Weapon_Mace`, `Weapon_Dagger` |
| `Armor_Bench` | Armor crafting | `Armor_Head`, `Armor_Chest`, `Armor_Legs`, `Armor_Hands` |
| `Cookingbench` | Food preparation | `Baked`, `Cooked`, `Prepared` |
| `Armory` | Advanced weapons | `Weapons.Sword`, `Weapons.Mace`, `Weapons.Bow` |
| `Forgery` | Smithing | Various |
| `Workbench` | General crafting | Various |

### Bench Tiers

Benches can have tier levels that unlock more advanced recipes:

```json
{
  "BenchRequirement": [
    {
      "Id": "Weapon_Bench",
      "Type": "Crafting",
      "RequiredTierLevel": 1
    }
  ]
}
```

## Best Practices

1. **Balance input costs** - Higher tier items should cost more
2. **Scale crafting time** - Complex items take longer
3. **Use appropriate benches** - Match bench to item type
4. **Consider knowledge** - Advanced recipes should require discovery
5. **Test resource availability** - Ensure ingredients are obtainable
6. **Use resource types** - Allow flexibility with fuel types
7. **Provide bench alternatives** - Multiple crafting paths when appropriate
